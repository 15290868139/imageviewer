支持左右/上下 滑动（基于ViewPager2）
支持常有图片操作手势（基于PhotoView）
支持查看超大图（基于SubsamplingScaleImageView）
支持简单的下拽退出
支持进入退出动画
支持数据内容异步追加.(向前翻页.聊天/帖子历史图片查看等)


使用简单. 调用 `ImageViewerBuilder.show` 即可
